## 🎯 Topics 配置策略

### 在 GitHub 仓库设置 Topics

1. 访问您的仓库页面
2. 点击 ⚙️ Settings
3. 滚动到 "Topics" 部分
4. 点击 "Add topics"
5. 从推荐列表中选择或输入以下标签

## 📝 Topics 使用规则

### GitHub Topics 规则

根据 [GitHub 文档](https://docs.github.com/en/articles/classifying-your-repository-with-topics)：

- ✅ 使用小写字母
- ✅ 可以包含数字
- ✅ 可以包含连字符（-）
- ✅ 每个标签最多 50 个字符
- ✅ 每个仓库最多 20 个标签
- ❌ 不要使用空格
- ❌ 不要使用特殊字符（除连字符外）

### 示例

```
✅ telegram-bot
✅ model-context-protocol
✅ python-3

❌ Telegram Bot
❌ model_context_protocol
❌ python3.11
```
